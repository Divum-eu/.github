# Divum - Free Minecraft Server Hosting

Divum is an open-source Minecraft server hosting service, aimed at people who'd rather not bother with self-hosting, configuration and maintenance. Head on over to [docs.divum.eu](docs.divum.eu) for documentation!

*As of now, Divum is being developed for a university course project assignment, but we are committed to the long-term development and support, and eventual public release for folks to use.*
